## 2. 텔레그램+카카오 봇 역할별 명령어

### 모니터링
- `상태` — 서버 상태 리포트
- `트랙` — 최근 트랙 목록
- `유저` — 유저 통계
- `댓글` — 최근 댓글
- `배포` — 사이트 접근 확인

### 관리
- `공지/공지삭제` — 인앱 공지 관리
- `삭제/공개/비공개` — 트랙 관리
- `댓글삭제` — 댓글 관리
- `알림` — 전체 푸시 발송

### 개발
- `수정 <지시>` — AI 코드 수정 (GitHub Issue → Claude Code Action)
- `PR` — PR 목록 / `머지 <번호>` — PR 머지
- `QA` — 전체 코드 점검 → 결과 리포트
- `진행상황` — GitHub Actions 실시간 추적

### 기획
- `기획 <기능>` — Issue 등록 (체크리스트 포함)
- `백로그` — 미완료 Issue 목록
- `버그 <설명>` — 버그 리포트

### 디자인
- `디자인 <지시>` — CSS/UI 수정 → Claude 자동 처리

### 사용량
- `사용량` — 전체 통계 대시보드
- `일간` — 오늘 활동 리포트
- `주간` — 7일 차트 리포트

## 3. GitHub Actions 워크플로우

| 워크플로우 | 기능 |
|-----------|------|
| `claude-fix.yml` | Issue → Claude Code Action → PR → 알림 (텔레그램+카카오) |
| `notify-pr.yml` | PR 생성 시 텔레그램+카카오 알림 |
| `notify-deploy.yml` | push → Vercel 배포 → 텔레봇 알림 |
| `e2e-daily.yml` | 매일 09:00 KST Playwright E2E 테스트 → 봇 리포트 |
