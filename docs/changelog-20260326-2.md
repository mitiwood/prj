# Changelog 2026-03-26 (2차)

## 닉네임 시스템 개선 + 커뮤니티 자동공개 + kie.ai API 대응 + 어시스턴트 수정

---

### 1. 닉네임 변경 즉시 반영 + DB 저장
- **파일**: `index.html`, `api/profile.js`
- 닉네임 저장 후 `renderSettings()` + `updateLoginUI()` 호출하여 설정/헤더/로그인 버튼 모두 즉시 갱신
- `api/profile.js`: 기존 `ua` 데이터 조회 후 merge하여 bio 외 데이터 유실 방지

### 2. 아바타 URL 텍스트 노출 버그 수정
- **파일**: `index.html`
- 카카오 등 이미지 URL 아바타를 `<img>` 태그로 렌더링
- `settings-avatar`, `login-area`, 로그인 버튼 3곳 수정
- URL이면 이미지 표시, 로드 실패 시 이름 첫글자 폴백

### 3. 닉네임 최대 10자 제한 + 안내 UI
- **파일**: `index.html`, `api/profile.js`
- 클라이언트 input `maxlength="10"` + 서버 `.slice(0, 10)` 이중 검증
- placeholder "최대 10자", 글자수 카운터 N/10, 10자 도달 시 빨간 경고 메시지

### 4. 로그인 버튼 닉네임 중복 표시 수정
- **파일**: `index.html`
- 아바타 첫글자 + 이름이 중복 표시되던 문제 수정 (예: "마 마빈" → "마빈 (로그아웃)")

### 5. 프로필 수정 Rate Limit 오탐 수정
- **파일**: `api/profile.js`
- IP 단독 → 사용자명+프로바이더+IP 조합으로 식별 변경
- 3회/분 → 10회/분 완화

### 6. 관리자 커뮤니티 자동/수동 공개 모드 제어
- **파일**: `api/app-settings.js` (신규), `admin/admin.html`, `api/tracks.js`, `index.html`
- Supabase `settings` 테이블에 `community_auto_publish` 설정 저장/로드
- 관리자 페이지에서 토글로 자동(생성 즉시 공개) / 수동(직접 공유 시만 공개) 전환
- 클라이언트에서 `_commAutoPublish` 플래그로 `_is_public` 결정

### 7. 사용자 식별자 email+provider 전환
- **파일**: `api/users.js`, `api/profile.js`, `api/tracks.js`, `index.html`, `api/supabase-schema.sql`, `api/migration-email-key.sql` (신규)
- users 테이블 UNIQUE 제약: `(name, provider)` → `(email, provider)` partial unique index
- tracks 테이블: `owner_email` 컬럼 추가
- 로그인/프로필 수정/트랙 저장/프로필 조회 시 email 기반 매칭
- 닉네임 변경해도 동일 사용자로 인식 (관리자 페이지 중복 해소)
- Supabase 마이그레이션 실행 완료 (중복 사용자 병합 + 기존 트랙 email 채우기)

### 8. 크리에이터 프로필 모든 생성곡 노출
- **파일**: `api/profile.js`
- 프로필 트랙 쿼리에서 `is_public=eq.true` 필터 제거

### 9. 로컬→서버 역방향 트랙 동기화
- **파일**: `index.html`
- `_syncUserDataFromServer()`에 역방향 동기화 로직 추가
- 서버에 없는 로컬 트랙을 자동 업로드 (최대 20개)
- 로그인 시/앱 복귀 시 자동 실행

### 10. 커뮤니티 목록 모든 생성곡 노출
- **파일**: `api/tracks.js`, `api/my-feed.js`, `api/daily-discover.js`
- 커뮤니티 피드/팔로잉 피드/오늘의 발견에서 `is_public` 필터 제거
- `audio_url` 존재 여부만 체크

### 11. kie.ai API 응답 구조 변경 대응
- **파일**: `index.html`, `api/callback.js`
- `streamAudioUrl` 필드를 `audioUrl` 폴백으로 추가
- `TEXT_SUCCESS` 상태를 완료 상태로 처리
- `audioUrl`이 비어있어도 `streamAudioUrl`로 재생 가능
- pollResult에서 유효 트랙(audioUrl 있는) 필터 적용

### 12. 케니 음악 어시스턴트 수정
- **파일**: `api/claude-chat.js`, `index.html`
- 타이틀: "Claude 음악 어시스턴트" → "케니 음악 어시스턴트", 아바타 C → K
- Anthropic Claude Haiku 전용 (kie.ai Gemini 폴백 제거로 속도 개선)
- Vercel에 `ANTHROPIC_API_KEY` 설정 확인 완료
