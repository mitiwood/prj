# Anthropic API 가이드

## 크레딧 충전

- **콘솔**: https://console.anthropic.com/settings/billing
- **최소 충전**: $5 (약 7,000원)
- **결제 수단**: Visa, Mastercard (신용/체크카드)

## 사용 모델

현재 Claude 챗(`/api/claude-chat`)에서 사용 중인 모델:

| 모델 | ID | 용도 |
|---|---|---|
| Claude Haiku 4.5 | `claude-haiku-4-5-20251001` | AI 채팅 어시스턴트 |

## 토큰 비용 (Claude Haiku 4.5)

| 항목 | 100만 토큰당 | 원화 환산 |
|---|---|---|
| 입력 (Input) | $1.00 | ~1,400원 |
| 출력 (Output) | $5.00 | ~7,000원 |

## 1회 대화 예상 비용

| 항목 | 토큰 수 | 비용 |
|---|---|---|
| 시스템 프롬프트 | ~300 | ~$0.0003 |
| 사용자 질문 | ~50 | ~$0.00005 |
| AI 응답 | ~200 | ~$0.001 |
| **합계** | **~550** | **~$0.0013 (약 2원)** |

## 충전 금액별 예상 사용량

| 충전 금액 | 예상 대화 횟수 |
|---|---|
| $5 (7,000원) | ~3,800회 |
| $10 (14,000원) | ~7,600회 |
| $25 (35,000원) | ~19,000회 |

## 레이트 리밋

- 유저당 **분당 10회** 제한 (`/api/claude-chat` 내부 구현)
- 대화 히스토리 **최대 20개** 메시지

## 다른 Claude 모델 비용 참고

| 모델 | 입력 (100만 토큰) | 출력 (100만 토큰) |
|---|---|---|
| Haiku 4.5 | $1.00 | $5.00 |
| Sonnet 4.6 | $3.00 | $15.00 |
| Opus 4.6 | $15.00 | $75.00 |

> Haiku 4.5가 가장 비용 효율적. 채팅용으로 적합.

