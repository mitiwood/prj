## 6. 데이터 흐름

```
사용자 액션
    ↓
클라이언트 검증 (플랜/크레딧/게스트 한도)
    ↓
API 호출 (/api/*)
    ↓
서버 검증 (인증/권한/플랜)
    ↓
Supabase (DB / Realtime / Storage)
    ↓
응답 → UI 업데이트
    ↓
알림 (Telegram / Kakao / Push)
```
