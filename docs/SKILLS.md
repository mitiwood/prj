# 스킬(슬래시 커맨드) 설명서

> Claude Code에서 `/스킬명` 으로 실행합니다.
> 인자가 필요한 스킬은 `/스킬명 인자` 형태로 사용합니다.

---

## 배포/운영

| 스킬 | 명령어 | 설명 |
|------|--------|------|
| 핫픽스 | `/hotfix` | 긴급 버그 수정 → 문법 검사 → 커밋 → 푸시 → 배포 |
| 셀프힐 | `/selfheal` | 배포 후 헬스체크 → 실패 시 자동 진단 + 수정 + 재배포 |
| 롤백 | `/rollback` | 이전 커밋으로 즉시 롤백 배포 |
| 스냅샷 | `/snapshot` | 현재 상태를 태그로 저장 (버전 백업) |
| 버전 목록 | `/versions` | 저장된 스냅샷 목록 조회 |
| 백업 | `/backup` | 주요 파일 스냅샷 백업 (대형 변경 전 안전장치) |

### 사용 예시
```
/hotfix 커스텀 모드 생성 버튼 iOS에서 안 됨
/rollback
/snapshot v2.1-stable
/versions
```

## 점검/모니터링

| 스킬 | 명령어 | 설명 |
|------|--------|------|
| 라이브 체크 | `/check-live` | ddinggok.com + API 엔드포인트 헬스체크 |

| 리뷰 | `/review` | 코드 리뷰 + 보안 점검 |
| 성능 | `/perf` | 사이트 성능 분석 |
| 정리 | `/cleanup` | 레거시 코드 탐지 + 불필요 파일 정리 |

### 사용 예시
```
/check-live
/qa
/report 오늘 수정한 버그 3건 정리해서 보내줘
/review index.html
```

## 효율적인 일괄 요청 방법

별도 스킬 없이 **한 메시지에 번호 리스트**로 보내면 됩니다:

```
다음 5가지 수정해줘:

1. [버그] iOS 커스텀 모드 생성 안 됨 — 에러: checkPlanLimit false
2. [버그] YouTube URL 붙여넣기 후 분석 안 됨 — 디바이스: 아이폰
3. [UI] QR 팝업 MZ스럽게 리디자인
4. [기능] 관리자 플랜 변경 시 확인 팝업 추가
5. [기능] 재생불가 곡 비활성화 + 안내 토스트
```

### 요청 작성 팁

| 유형 | 포맷 |
|------|------|
| 버그 | `[버그] 증상 — 에러메시지 — 디바이스` |
| UI | `[UI] 어떤 화면 + 무엇을 + 어떻게` |
| 기능 | `[기능] 한 줄 설명` |
| 성능 | `[성능] 어디가 느린지 + 체감 시간` |
