소셜 로그인 연동 방법
Google
1. console.cloud.google.com → 프로젝트 생성
2. API 및 서비스 → OAuth 2.0 클라이언트 ID 생성
3. 승인된 리디렉션: https://도메인/api/auth/google/callback
4. Vercel 환경변수: GOOGLE_CLIENT_ID, GOOGLE_CLIENT_SECRET
Kakao
1. developers.kakao.com → 내 애플리케이션 → 앱 추가
2. 카카오 로그인 → Redirect URI: https://ai-music-studio-bice.vercel.app/api/auth/kakao/callback
3. Vercel 환경변수: KAKAO_CLIENT_ID, KAKAO_CLIENT_SECRET
Naver
1. developers.naver.com → Application → 애플리케이션 등록
2. 서비스 URL + Callback URL: /api/auth/naver/callback
3. Vercel 환경변수: NAVER_CLIENT_ID, NAVER_CLIENT_SECRET
💡 NextAuth.js 또는 직접 OAuth flow 구현
