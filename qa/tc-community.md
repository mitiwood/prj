# TC — 커뮤니티 기능

> 우선순위: P1 = 필수 / P2 = 중요 / P3 = 선택  
> 업데이트: 2026-04-01

---

## 좋아요 / 싫어요

### TC-C01 · P1 · 좋아요 추가
**전제조건**: 로그인, 공개 트랙  
**스텝**:
1. 트랙 카드 ❤️ 버튼 클릭  

**기대결과**: 카운트 +1, 버튼 빨간색으로 변경, DB `comm_likes` 업데이트

---

### TC-C02 · P1 · 좋아요 취소 (toggle)
**스텝**: 이미 좋아요한 트랙 ❤️ 재클릭  

**기대결과**: 카운트 -1, 버튼 원래 색상 복귀

---

### TC-C03 · P2 · 좋아요 + 싫어요 동시 방지
**스텝**: 좋아요 상태에서 싫어요 클릭  

**기대결과**: 좋아요 자동 취소 후 싫어요 적용, 두 카운트 각각 올바르게 반영

---

### TC-C04 · P2 · 비로그인 좋아요 시도
**전제조건**: 로그아웃 상태  
**스텝**: 좋아요 버튼 클릭  

**기대결과**: 로그인 유도 모달 표시, 좋아요 미처리

---

### TC-C05 · P2 · 좋아요 순위 반영
**스텝**: 좋아요 후 커뮤니티 → 인기순 정렬  

**기대결과**: 좋아요 많은 트랙 상단 노출 (실시간 또는 일정 딜레이 후)

---

## 댓글

### TC-C06 · P1 · 댓글 작성
**전제조건**: 로그인  
**스텝**:
1. 트랙 상세 → 댓글 입력
2. 전송  

**기대결과**: 댓글 목록 최상단에 추가, 작성자 이름/아바타 표시, Telegram/Kakao 알림 발송

---

### TC-C07 · P1 · 댓글 목록 조회
**스텝**: 댓글 있는 트랙 상세 열기  

**기대결과**: `is_hidden=false`인 댓글만 표시, 최신순, 작성자/시간 표시

---

### TC-C08 · P2 · 대댓글 (답글)
**스텝**: 댓글 → 답글 버튼 → 내용 입력 → 전송  

**기대결과**: 부모 댓글 아래 들여쓰기로 표시, `parent_id` 연결 확인

---

### TC-C09 · P2 · 댓글 300자 제한
**스텝**: 300자 초과 텍스트 입력 후 전송  

**기대결과**: 입력 차단 또는 에러 메시지, API에 요청 미전송

---

### TC-C10 · P2 · 빈 댓글 전송 방지
**스텝**: 공백만 입력 후 전송  

**기대결과**: 전송 버튼 비활성 또는 에러

---

### TC-C11 · P3 · 댓글 Rate Limit
**스텝**: 1분 내 10회 이상 댓글 전송  

**기대결과**: 429 또는 에러 토스트 표시, 이전 댓글 보존

---

## 팔로우

### TC-C12 · P1 · 팔로우/언팔로우
**스텝**:
1. 크리에이터 프로필 → 팔로우 버튼  
2. 재클릭 → 언팔로우  

**기대결과**: 팔로워 카운트 +1/-1, 버튼 상태 전환, DB `follows` 테이블 반영

---

### TC-C13 · P2 · 자기 자신 팔로우 방지
**스텝**: 본인 프로필에서 팔로우 버튼 확인  

**기대결과**: 팔로우 버튼 숨김 또는 비활성

---

### TC-C14 · P2 · 팔로잉 피드 반영
**스텝**:
1. 크리에이터 A 팔로우
2. 피드 탭 새로고침  

**기대결과**: A의 최근 트랙이 피드에 표시

---

## 크리에이터 프로필

### TC-C15 · P1 · 프로필 기본 정보 표시
**스텝**: 크리에이터 이름 클릭 → 프로필 진입  

**기대결과**: 닉네임, 아바타, 트랙 수, 총 좋아요, 팔로워/팔로잉 수 표시

---

### TC-C16 · P2 · 크리에이터 레벨 뱃지
**스텝**: 다양한 레벨 크리에이터 프로필 확인  

**기대결과**: rookie/rising/star/legend 뱃지 기준에 맞게 표시

---

### TC-C17 · P2 · 프로필 이름 수정
**스텝**: 내 프로필 → 이름 편집 → 저장  

**기대결과**: 표시 이름 변경, 기존 트랙 카드의 작성자명 동기화

---

## 커뮤니티 피드

### TC-C18 · P1 · 피드 기본 로딩
**스텝**: 커뮤니티 탭 진입  

**기대결과**: 공개 트랙 목록 표시, 각 카드에 썸네일/제목/좋아요 수 표시

---

### TC-C19 · P1 · 정렬 변경 (최신/인기/별점)
**스텝**: 커뮤니티 → 정렬 탭 전환  

**기대결과**: 각 정렬 기준에 따라 순서 변경, 로딩 표시 후 결과 업데이트

---

### TC-C20 · P2 · 내 피드 (팔로잉)
**스텝**: 피드 탭 → "팔로잉" 필터  

**기대결과**: 팔로우한 크리에이터의 트랙만 표시, 미팔로우 트랙 제외

---

### TC-C21 · P2 · 무한 스크롤 / 페이징
**스텝**: 피드 스크롤을 끝까지 내리기  

**기대결과**: 다음 페이지 자동 로드, 스피너 표시 후 추가 트랙 노출

---

---

### TC-C24 · P2 · 삭제된 트랙 공유 링크 접근
**스텝**: 삭제된 트랙의 공유 URL 접근  

**기대결과**: 404 또는 "트랙을 찾을 수 없습니다" 화면, 앱 크래시 없음

## 콜라보레이션

### TC-C32 · P2 · 콜라보 요청 전송
**스텝**: 크리에이터 프로필 → 콜라보 요청 → 메시지 입력 → 전송  

**기대결과**: 상대방 인박스에 요청 도착, 요청자에게 "전송됨" 표시

---

### TC-C33 · P2 · 콜라보 수락/거절
**스텝**: 인박스 → 요청 수락 또는 거절  

**기대결과**: 수락 시 콜라보 워크스페이스 생성, 거절 시 요청자에게 알림
