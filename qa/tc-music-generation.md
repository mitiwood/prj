# TC — 음악 생성

> 우선순위: P1 = 필수 / P2 = 중요 / P3 = 선택  
> 업데이트: 2026-04-01

---

## 커스텀 모드

### TC-M01 · P1 · 기본 생성 (가사 + 스타일)
**전제조건**: 로그인, 크레딧 1개 이상  
**스텝**:
1. 커스텀 탭 선택
2. 가사 입력 (최소 2행 이상)
3. 스타일 태그 2개 이상 선택
4. 생성 버튼 클릭  

**기대결과**: 결과 배치에 카드 1개 생성, 오디오 URL 존재, 크레딧 1 차감  
**엣지케이스**: 가사 5,000자 초과 시 입력 차단

---

### TC-M02 · P1 · 가사 없이 생성 (instrumental)
**스텝**:
1. 커스텀 탭, 가사 비움
2. 인스트루멘탈 옵션 ON
3. 생성  

**기대결과**: 가사 없이 배경음 트랙 생성, 가사 탭 "가사 없음" 표시

---

### TC-M03 · P2 · 섹션 태그 파싱
**스텝**:
1. 가사에 `[Verse]`, `[Chorus]`, `[Bridge]` 포함
2. 생성 후 가사 탭 확인  

**기대결과**: 섹션 태그가 구분선/헤더로 렌더링, 원본 태그 그대로 보존

---

### TC-M04 · P2 · 제목 자동 생성
**스텝**:
1. 제목 입력 없이 생성  

**기대결과**: AI가 가사/프롬프트 기반으로 제목 자동 부여, 카드 상단에 표시

---

### TC-M05 · P1 · 크레딧 부족 시 차단
**전제조건**: 크레딧 0개  
**스텝**: 생성 버튼 클릭  

**기대결과**: 크레딧 부족 토스트/모달, 생성 미시작, API 호출 없음

---

## 심플 모드

### TC-M06 · P1 · 간단 설명으로 생성
**스텝**:
1. 심플 탭 선택
2. 한 줄 설명 입력 (예: "밝고 신나는 여름 팝송")
3. 생성  

**기대결과**: AI가 가사 자동 생성 후 트랙 완성, `lyrics-plain` 영역에 가사 채워짐

---

### TC-M07 · P2 · 가사 자동완성 타이밍
**스텝**:
1. 심플 모드로 생성
2. 생성 완료 직후 가사 탭 클릭  

**기대결과**: 가사 없으면 "생성 중..." 스피너, 500ms 후 자동 채워짐 (없으면 "가사 없음")

---

### TC-M08 · P2 · 짧은 설명 (1단어)
**스텝**: 설명 입력란에 "슬픔" 한 단어만 입력 후 생성  

**기대결과**: 정상 생성, 에러 없음

---

## YouTube 모드

### TC-M09 · P1 · 유효한 YouTube URL 분석
**스텝**:
1. YouTube 탭 선택
2. 유효한 YouTube URL 붙여넣기
3. 분석 → 생성  

**기대결과**: 장르/분위기/BPM 태그 추출, 스타일 기반 트랙 생성

---

### TC-M10 · P1 · 잘못된 URL 입력
**스텝**: YouTube URL 아닌 임의 URL 입력 후 분석  

**기대결과**: 에러 토스트 "유효하지 않은 YouTube URL", 생성 진행 안 됨

---

### TC-M11 · P2 · 비공개 영상 URL
**스텝**: 비공개/삭제된 YouTube 영상 URL 입력  

**기대결과**: "영상을 불러올 수 없습니다" 에러 처리, 앱 정상 유지

---

## MV 모드

### TC-M12 · P1 · 기존 트랙으로 MV 생성
**전제조건**: 히스토리에 트랙 1개 이상  
**스텝**:
1. MV 탭 → 트랙 선택
2. 영상 스타일 프롬프트 입력
3. 생성  

**기대결과**: 영상 URL 포함된 MV 카드 생성, MV 크레딧 차감

---

### TC-M13 · P2 · MV 생성 중 상태 표시
**스텝**: MV 생성 시작 후 로딩 상태 관찰  

**기대결과**: 진행률 또는 "생성 중..." 인디케이터 표시, 완료 시 카드에 영상 재생 가능

---

## 연장 (Extend)

### TC-M14 · P1 · 기본 연장
**스텝**:
1. 결과 카드 → 🛠 도구 → 연장
2. 연장 위치 슬라이더 조정
3. 연장 프롬프트 입력
4. 연장 실행  

**기대결과**: 원본보다 긴 트랙 생성, 히스토리에 별도 항목 추가

---

### TC-M15 · P2 · 연장 가사 입력
**스텝**: 연장 모달에서 추가 가사 입력 후 연장  

**기대결과**: 입력 가사가 연장 구간에 반영됨

---

### TC-M16 · P3 · 연장 위치 경계값
**스텝**: 슬라이더 최솟값(0), 최댓값(300)으로 각각 연장  

**기대결과**: 에러 없이 정상 처리

---

## 다시 작곡

### TC-M17 · P1 · 다시 작곡 기본 동작
**스텝**:
1. 결과 카드 → 🔄 다시 작곡
2. 확인  

**기대결과**: 동일 입력값으로 새 트랙 생성, 기존 카드 유지, 크레딧 차감

---

## A/B 비교

### TC-M18 · P1 · 2곡 동시 생성 레이아웃
**전제조건**: A/B 비교 모드 or 2곡 이상 생성  
**스텝**: 2곡 생성  

**기대결과**: 배치 헤더에 "A/B 비교" 뱃지, 각 카드 커버에 A/B 뱃지 표시, "A 버전 선택" / "B 버전 선택" 버튼 표시

---

### TC-M19 · P1 · A/B 버전 선택
**스텝**:
1. A 버전 선택 버튼 클릭  

**기대결과**: A 카드 체크 표시, B 카드 흐려짐, 라이브러리로 이동 후 해당 트랙 최상단

---
