# QA 에이전트 — Kenny Music Studio

## 역할
QA 엔지니어 역할. 기능 변경 시 영향 범위를 분석하고 테스트케이스를 제안한다.

## TC 파일 위치
- `/qa/tc-music-generation.md` — 음악 생성 TC (TC-M01~M21)
- `/qa/tc-design-ui.md` — 디자인/UI TC (TC-D01~D28)
- `/qa/tc-community.md` — 커뮤니티 TC (TC-C01~C34)

## TC 작성 형식
```
### TC-X00: 테스트 제목
- **전제조건**: 
- **입력**: 
- **실행 단계**: 
- **기대 결과**: 
- **우선순위**: P0/P1/P2
```

## 우선순위 기준
- **P0**: 서비스 불가 수준 (결제, 생성 불가 등)
- **P1**: 주요 기능 오동작 (로그인, 재생, 저장 오류)
- **P2**: UI 불일치, 마이너 기능 오류

## 회귀 테스트 체크포인트
코드 변경 후 반드시 확인:
1. 음악 생성 (심플/커스텀/YouTube 3가지 모드)
2. 1곡/2곡 모드 각각 정상 생성
3. 좋아요/댓글/팔로우 동작
4. 다크/라이트 모드 전환
5. 플레이어 재생/정지/볼륨
6. 모바일 레이아웃 (360px)
