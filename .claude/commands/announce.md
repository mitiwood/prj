공지사항 관리 스킬. CLI에서 공지를 등록, 수정, 삭제한다.

**Instructions:**
1. `$ARGUMENTS`가 없으면 사용 가능한 명령어 목록을 보여준다:
   - `list` — 현재 공지사항 목록 조회
   - `add <내용>` — 새 공지 등록
   - `edit <ID> <내용>` — 공지 수정
   - `delete <ID>` — 공지 삭제
   - `status` — 공지 API 상태 확인

2. `list` 명령:
   - `api/announcement.js`의 데이터 소스 확인
   - Supabase 또는 파일 기반 데이터 조회
   - 공지 목록 테이블 형식으로 표시

3. `add` 명령:
   - 공지 내용을 받아 API를 통해 등록
   - 등록 후 확인 메시지 표시

4. `edit` 명령:
   - ID로 기존 공지 찾기
   - 내용 수정 후 저장

5. `delete` 명령:
   - 삭제 전 해당 공지 내용 표시
   - 사용자 확인 후 삭제

6. `status` 명령:
   - `/api/announcement` 엔드포인트 응답 테스트
   - 데이터 소스 연결 상태 확인

**Rules:**
- 공지 등록/수정/삭제는 사용자 확인 후 실행
- API 엔드포인트 구조에 맞게 요청 형식 자동 조정
- 인증이 필요한 경우 안내 메시지 표시
