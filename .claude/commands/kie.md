kie.ai API 레퍼런스 조회. 자연어 질문으로 KIE_API_REFERENCE.md에서 해당 섹션을 찾아 답변한다.

사용자 질문: $ARGUMENTS

**Instructions:**

1. `KIE_API_REFERENCE.md` 파일을 읽는다.

2. 사용자 질문(`$ARGUMENTS`)을 분석하여 매칭되는 섹션을 찾는다:
   - 번호(예: "3", "2.1") → 해당 섹션 직접 출력
   - "음악" / "생성" / "generate" → 섹션 2
   - "가사" / "lyrics" → 섹션 3
   - "연장" / "extend" → 섹션 4
   - "보컬" / "vocal" → 섹션 5
   - "타임스탬프" / "카라오케" → 섹션 6
   - "폴링" / "상태" / "record-info" → 섹션 7
   - "에러" / "오류" / "코드" → 섹션 8
   - "모델" / "model" → 섹션 9
   - "콜백" / "webhook" → 섹션 10
   - "가격" / "크레딧" / "인증" → 섹션 1
   - 인자 없으면 → 전체 섹션 목록 출력

3. 해당 섹션의 **전체 내용**을 그대로 출력한다 (엔드포인트, 파라미터 테이블, 예시 코드 포함).

4. 질문이 여러 섹션에 걸치면 관련 섹션 모두 출력한다.

5. 문서에 없는 내용이면: "KIE_API_REFERENCE.md에 없는 정보입니다. https://docs.kie.ai/ 에서 확인하세요."

**Rules:**
- KIE_API_REFERENCE.md 내용만 기반으로 답변
- API 키/시크릿 절대 출력 금지
- 요약하지 말고 원문 그대로 출력
